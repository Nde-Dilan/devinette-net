# **App Name**: DevinetteNet

## Core Features:

- Play Riddle: Fetch a random riddle from Firestore and display it, allowing users to guess and reveal the answer.
- Submit Riddle: Provide a form for users to submit new riddles, including question, answer, origin, and language, and save them to Firestore.
- Explore Riddles: List all riddles from Firestore with the ability to filter by origin or language, and reveal answers on tap.
- Anonymous Authentication: Enable anonymous authentication to allow users to submit riddles without creating an account.
- Random Riddle Selection: Implement an AI tool to improve the randomness and/or adaptiveness of the selection of riddles presented to the user based on a suitability score (such as previously attempted, user origin, etc.)
- Share Riddle: Add a share button to allow users to share riddles on social media.

## Style Guidelines:

- Primary color: Deep Orange (#FF7043), evokes warmth and African themes.
- Background color: Light Orange (#FFCCBC), creates a bright and welcoming atmosphere.
- Accent color: Brown (#D76823), complements the warm tones and provides visual contrast for key UI elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font that balances modernity and warmth, ensuring readability and approachability.
- Use local asset icons or emojis that align with African themes and riddle categories to add a playful touch.
- Employ Material 3 widgets to ensure a simple and mobile-friendly layout.
- Implement subtle animations to enhance user experience, especially when revealing answers or transitioning between riddles.
